# 三国群英传2威力加强版复原说明

## 已接入内容

1. 从 `三国群英传2威力加强版 v2.13.1.apk.1` 恢复出 `MOD06.xml`。
2. `MOD06.xml` 当前包含 21 个势力、94 个城池、1055 名武将。
3. 城池保留 APK 中恢复出的名称、势力、太守/军师名、人口、金钱、城防、地图坐标、旗帜坐标。
4. 武将保留 APK 中恢复出的姓名、Face 编号、武力、智力、体力、技力、登场状态、当前城池、武器、坐骑、物品、兵种、阵型、武将技原文、军师技原文、人物说明。
5. 当前 Unity 运行时已改为按 MOD XML 动态初始化势力、城池、武将数量，不再固定 18/48/255。

## 兼容处理

1. 当前一代引擎没有二代全部兵种、阵型、装备、技能资源，`Sango2RestoreManifest.tsv` 记录了每一项兼容转换或未装配原因。
2. 预备兵上限未在已恢复 APK 城池表中找到，当前按人口和金钱派生为引擎可运行值，并在清单中标记为 `not_recovered`。
3. 旧 48 城仍使用原路径拓扑；新增城池按恢复坐标生成近邻或直达路径。
4. 当前仓库只有 `Head001` 到 `Head255` 头像 prefab。MOD06 会保留二代 Face 编号，缺少头像资源时输出错误日志，不串用假头像。
5. 战斗小人资源仍只有一代 255 套映射。超过范围的武将会输出错误日志，并使用基础部件防止数组越界。

## 仍需继续复原

1. 二代完整头像 prefab、战斗小人、旗帜、技能、兵种和阵型资源。
2. 二代完整地图贴图与路径拓扑。
3. APK IL2CPP 逻辑层仍未完整反编译成功；当前是基于可恢复数据与现有一代引擎结合的可运行版本。
